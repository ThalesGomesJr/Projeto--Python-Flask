||Ativar Anaconda-Python3.7: conda config --set auto_activate_base true||
||Desativar Anaconda-Python3.7: conda config --set auto_activate_base false||

||Instalar pipenv: pip install pipenv||
||Criar ambiente virtual: pipenv --three||
||Verificar dependencias: pipenv run pip freeze||
||Abrir ambiente virtual: pipenv shell||

||Criar requirements: pip freeze > requirements.txt||
||Instalar requirements: pipenv install -r requirements.txt||
