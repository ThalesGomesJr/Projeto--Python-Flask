||Ativar Anaconda-Python3.7: conda config --set auto_activate_base true||
||Desativar Anaconda-Python3.7: conda config --set auto_activate_base false||
||Ativar Anaconda-Python3.7: conda activate||
||Desativar Anaconda-Python3.7: conda deactivate||

||Instalar virtualenv: pip install virtualenv||
||Criar ambiente virtual: virtualenv venv||
||Verificar dependencias: pip freeze||
||Abrir ambiente virtual: source venv/bin/activate||

||Criar requirements: pip freeze > requirements.txt||
||Instalar requirements: pip install -r requirements.txt||
